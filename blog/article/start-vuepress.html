# VuePressを利用してブログを導入してみた

# GitHub PagesとVuePress

これまで、ポートフォリオサイトの運用を"gulpを利用したWebサイト制作用のコードベース"でおこなってきましたが、更新する際に手間が掛かってしまうことを懸念して、この度、GitHub PagesプラスVuePressでのリニューアルをおこないました。

GitHub PagesはGitHubのリポジトリを利用したホスティングサービスで、特定の命名ルールでリポジトリを作成するか、特定の名前のブランチを作成するとそのリポジトリをWebサイトとしてホストできます。

VuePressは、静的サイトジェネレーターと言うもので、有名なものではJekyllやHugoなどが挙げられます。 特徴としては、Markdown記法でWebページを作成でき、簡単にサイトを構築できます。
前述のJekyllやHugoとの違いは、利用されるエンジンの違いです。VuePressはその名の通りVue.jsを利用してレンダリングします。

つまり、メモを作成するかのようにマークダウンファイルを作成し、リポジトリにpushするだけでサイトの更新ができます。
良き。

実を言うと、まだVuePressにはブログの機能がほとんど実装されておらず、実用的ではありません。
バージョンも執筆時点で0.14.8なので当然ではありますが、Vueで作られているため爆速でサイトを閲覧できるので利用しています。
今後に期待して更新情報をキャッチアップしつつこのサイトに反映できたらと考えています。
